# Audiobook Tool

[![python][python-src]][python-href]

Python CLI to handle audiobooks: build from MP3, extract from M4B, fusion MP3 to M4B, fetch metadata...

> [!IMPORTANT]
> Not ready for production.

## Using as CLI

```sh
pip install -e .
```

```sh
audiobook-tool build ./path/to/mp3_directory
```
